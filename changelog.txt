v0.2 changes:

    -   Фикс ссылок пагинации в shop_page.html - 
        {% url 'products' page %}{{request.get_full_path|cut:request.path}}
        Передается чистый GET запрос в следущую страницу.

    -   На главной странице сайта раньше получали все товары из БД и их сортировали. Но что если их будет тысячи? 
        Теперь для каждой категории получаем N-ое кол-во последних добавленных в БД товаров.

    -   Рандомный товар через order_by('?').first()

    -   Личный кабинет, система авторизации, регистрации, смены пароля

    -   Страница с товарами с возможностью сортировки

    -   Скрипт генерации товаров DB_AUTO_FILL для демонстрации работы сайта
        (Доступно только админам)

        .../db_auto_fill/7/Categories/
        .../db_auto_fill/10/Subcategories/
        .../db_auto_fill/300/Products/

    -   Отдельная встраиваимая HTML карточка товара

    -   У карточки товара появилась система изменения дизайна.
    
        "Новинка" если товар был размещен в БД в течение N дней.
        "Скидка" если у товара указана цена со скидкой.

    -   Добавлен адекватный футер

    -   "Докеризация" проекта

        # rm

            $ docker-compose -f docker-compose.prod.yml exec web python manage.py migrate --noinput
            $ docker-compose -f docker-compose.prod.yml exec web python manage.py collectstatic --no-input --clear

    -   Были добавлены хлебные крошки на основе наследования шаблонов

    -   Обновлен title страницы по аналогии с хлебными крошками

    -   Создана страница товара
